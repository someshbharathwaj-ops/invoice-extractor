# Invoice Extractor Frontend

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- React Hook Form + Zod
- TanStack Table
- Recharts
- Radix / ShadCN-style UI primitives

## Run

```bash
cd frontend
npm install
npm run dev
```

## Notes

- Frontend is isolated from the Python extraction pipeline.
- Current UI uses demo data and a simulated processing flow until a dedicated API layer is wired in.
